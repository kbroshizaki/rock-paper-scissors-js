# rock-paper-scissors-js
I built rock paper scissors for 5 rounds against a computer.

a command line game of rock, paper, scissors built with javascript and node.js.

Play 5 rounds against a computer
track score across rounds
announces winner for each game and overall winner at the end of 5 rounds
uses modular JS files

Technology:
- JavaScript
- Node.js
- prompt-sync

Installation:
npm install

run:
node playgame.js
